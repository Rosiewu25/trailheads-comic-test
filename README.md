# Comics Run on Context

A one-page interactive "paper app" built with a single HTML file.

## Description

This project explores how readers interpret comic panels through context. The app presents a reaction panel before revealing the panel that came before it, encouraging readers to reconsider the same drawing after new information is introduced.

Rather than changing the artwork, the app changes the surrounding context to demonstrate how sequence influences interpretation.

## Features

- Single self-contained `index.html`
- No frameworks or build tools
- HTML, CSS, and JavaScript only
- Works on GitHub Pages

## How to Run

Open `index.html` in a web browser, or visit the GitHub Pages site after deployment.

## Project Goal

The goal is to explore how comic readers construct meaning from sequential images and how the same reaction panel can be interpreted differently depending on what precedes it.
